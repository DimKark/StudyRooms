# StudyRooms

Clone this repository:
* With the web URL:
```shell
    git clone https://github.com/DimKark/StudyRooms.git
```
* With SSH key:
 ```shell
    git clone https://github.com/DimKark/StudyRooms.git
```
---
Always run this before working on project
```shell
    git pull origin main
```
---

Run:
* MacOS / Linux
```shell
    ./mvnw spring-boot:run
```
* Windows
```shell
    ./mvnw.cmd spring-boot:run
```
