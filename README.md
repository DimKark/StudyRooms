# StudyRooms

Clone this repository:
```shell
git clone https://github.com/DimKark/StudyRooms.git
```
---
Always run this before working on project
```shell
git pull origin main
```
---
Run:
* MacOS / Linux
```shell
./mvnw spring-boot:run
```
* Windows
```shell
./mvnw.cmd spring-boot:run
```
---
### Requires the SRV Service

https://github.com/DimKark/SRVS
